# Sansvarg_NovaLearn
Gamified AI learning platform with multi-month premium access via PayPal Hosted 
Sansvarg NovaLearn is an AI-powered gamified learning platform. Users can unlock premium rewards by purchasing monthly access through PayPal Hosted Buttons, selecting how many months to pay for. A promo code (TEST123) allows free access for testing. The repository includes:
	•	index.html — landing page
	•	pricing.html — pricing & payment page
	•	dashboard.html — premium rewards page
	•	style.css — styling
	•	Fully functional PayPal integration with quantity-enabled hosted buttons
	•	Frontend promo code handling for testing
